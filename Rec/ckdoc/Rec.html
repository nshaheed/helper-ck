Rec

This Rec class provides helper functionality for recording to audio files. It supports recording from dac, ugens, and arrays of ugens.

Rec

inherits : Object

void auto()

Automatically record the DAC to a stereo file and store in the current directory. Will prepend the datetime to the file.

void auto(UGen ugen)

Automatically record a UGen and store in the current directory. Will prepend the datetime to the file and handle different numbers of channels automatically.

void auto(UGen[] ugen)

Automatically record an array of UGens and store in the current directory. Will prepend the datetime to the files (one file per UGen).

void autoMulti(UGen ugen, string dir)

Automatically record a multichannels UGen and store in specified directory. Will prepend the datetime to the files (one file per channel).

void autoStereo(UGen ugen, string dir)

Automatically record a stereo UGen and store in specified directory. Will prepend the datetime to the file.
